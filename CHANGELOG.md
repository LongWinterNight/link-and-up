# Changelog

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/); версии — [SemVer](https://semver.org/lang/ru/).

## [0.3.0] — 2026-07-14

### Добавлено
- Динамические кластеры тем из корпуса пользователя (мультиниша): TF-IDF + слияние микрокластеров, имя = топ-слова, редактор имён/ключевых слов, пересборка и сброс к встроенным. Схема состояния v3 с миграцией; бэкапы старых версий принимаются.
- Сравнение вариантов хука в «Сегодня»: до 3 альтернатив ранжируются между собой по диапазону, гардрейлы проверяются для каждого, «Сделать основным».
- Шаримая PNG-карточка разбора поста: генерация строго локально, явное согласие, обязательный водяной знак, автогенерируемый alt-текст.
- Двухслойная подача прогноза: по умолчанию — одна человеческая строка и диапазон, MAPE/бэктест/разложение — под раскрытием «для эксперта».
- Индикатор разнообразия контент-плана (≥70% одной формулы за 30 дней — предупреждение).
- Полная локализация RU/EN: все вкладки, модалки, тосты; словарь меток enum-значений с фолбэком.
- e2e-смоук (Playwright, прод-сборка), Lighthouse на PR, перф-гейт 20K постов в CI; Prettier с format-check.
- Индикатор занятого хранилища в настройках; LSH-префильтр near-dup для быстрых импортов больших личных корпусов.

## [0.2.0] — 2026-07-13

### Добавлено
- Экран «Сегодня» — новый дефолт: идея → черновик по формуле → диапазон вовлечения → гардрейлы → действия; быстрая идея инлайн.
- Импорт официального экспорта LinkedIn (Shares.csv) и клип-форма одного поста-референса.
- Эмпирические множители прогноза из корпуса пользователя с автовыбором по leave-one-out бэктесту; перцентиль «выше, чем у X% корпуса».
- Нишевый пакет правил «Финтех» (черновик, не юрконсультация) и выбор ниши в онбординге/настройках.
- Локализация: инфраструктура RU/EN, переключатель, EN для главной вертикали.
- Бэкап/восстановление всего состояния одним JSON; экспорт журнала действий в CSV.
- Пресеты фильтров, табличный вид постов, увеличение scatter-графика в модалке.
- Публичное демо: https://link-and-up.vercel.app (security-заголовки, OG-карточки).

### Изменено
- Прогноз показывает диапазон как главное число (точка — вторичный «центр»), метку свежести корпуса; калибровка применяется от 3 фактов.
- Хранилище — IndexedDB (миграция из localStorage прозрачная, .bak-снапшот); отложенная запись без сериализации на каждое действие.
- Scatter — логарифмические оси с честными тиками; выборка с подписью на больших корпусах.
- Стор разобран на слайсы; вкладки — ленивые чанки (initial ≈90 КБ gzip).

### Исправлено
- Гонка гидратации, затиравшая мигрированные данные дефолтами.
- Слипание точек scatter на линейной шкале; фризы бэктеста (O(n²)→O(n log n), ×283 на 20K) и импорта (чанки+прогресс).
- ER больше не считается нулём при неизвестных комментариях; язык не определяется ложно по URL в начале текста.
- CSV formula injection, ReDoS пользовательских правил, лимиты импорта, редакция hard-терминов во всех экспортах.
- Из демо-корпуса удалены нереконструируемые пермалинки (провенанс) — источник ищется по точной цитате.

### Безопасность
- Формальный security-review без находок; CSP заголовками; шрифты self-hosted; ErrorBoundary с локальным логом.
