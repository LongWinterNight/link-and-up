# Политика безопасности

## Поддерживаемые версии

Поддерживается последний релиз на ветке `main` (демо: https://link-and-up.vercel.app).

## Сообщить об уязвимости

Пожалуйста, не открывайте публичные issue для уязвимостей. Используйте
[GitHub Security Advisories](https://github.com/LongWinterNight/link-and-up/security/advisories/new)
(Report a vulnerability) — это приватный канал.

Ответ — в течение 7 дней. После исправления уязвимость раскрывается в CHANGELOG.

## Модель безопасности (коротко)

- Приложение полностью клиентское: данные пользователя живут в IndexedDB/localStorage браузера
  и не отправляются на сервер. Внешних сетевых запросов из кода нет (шрифты self-hosted).
- CSP доставляется HTTP-заголовками (vercel.json) и meta-тегом прод-сборки.
- Пользовательские regex-правила проходят валидацию против ReDoS; импорт ограничен по размеру
  и числу записей; CSV-экспорт нейтрализует formula injection; URL с опасными схемами отбрасываются.
